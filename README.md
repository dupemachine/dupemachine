## Camera Viewer 
### Quick Start
- Clone the repo branch
- Navigate to /cameraviewer/ path
- Run the following in the cli:
```sh
git checkout cameraviewer-1
npm i
npm start
```
- Project will be running at localhost:4200

### Running unit tests
```sh
ng test
```
